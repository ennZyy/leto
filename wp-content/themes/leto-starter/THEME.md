# Описание темы Leto Starter

Справочный файл по теме `leto-starter` для ИИ-агентов. Описывает дизайн-систему,
структуру шаблонов и соглашения, чтобы при выполнении задач не нужно было
каждый раз перечитывать все файлы темы.

## Назначение

Классическая (не FSE) тема WordPress для лендинга коттеджного посёлка «ЛЕТО»
(Нижегородская область, Богородский район). Основной экран — одностраничный
лендинг (`front-page.php`), собранный из секций через `template-parts`.

## Дизайн-система

### Основные цвета

| Назначение            | Значение  | Переменная CSS                  |
|-----------------------|-----------|---------------------------------|
| Основной (тёмно-зелёный) | `#0E3D2E` | `--leto-color-primary`          |
| Ссылка (ховер)        | `#0a2e22` | `--leto-color-link-hover`       |
| Фон страницы          | `#f6f7f7` | `--leto-color-bg`               |
| Поверхность (карточки) | `#ffffff` | `--leto-color-surface`          |
| Граница               | `#dcdcde` | `--leto-color-border`           |
| Приглушённый текст    | `#50575e` | `--leto-color-muted`            |
| Основной текст        | `#1d2327` | `--leto-color-text`             |
| Кремовый фон секций   | `#F5EFD9` | `--leto-color-queues-bg`        |
| Голубой акцент (радио, eyebrow) | `#5BB6FF` | `--leto-color-ecofarm-radio`, `--leto-color-lake-eyebrow` |
| Синий акцент (застройщик) | `#1E40AF` | `--leto-color-developer-link`   |

Зелёный `#0E3D2E` — базовый брендовый цвет (хедер-кнопки, футер, hero,
очереди, глэмпинг, ecofarm). Кремовый `#F5EFD9` используется как фон
альтернирующих секций (очереди, ecofarm, застройщик).

### Шрифты

- Логотип/бренд: **Pacifico** (Google Fonts, подключается в `functions.php`
  только для кириллического «Лето»), фолбэк — `'Brush Script MT'`, cursive.
- Основной текст: системный стек
  `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", sans-serif`.
- Размер базового шрифта: `1rem`, межстрочный `1.6`.
- Заголовки используют `clamp()` для адаптивности (например, hero-title
  `clamp(2.25rem, 4.5vw, 3.75rem)`).

### Сетка и адаптив

- Контейнер: `.leto-container`, `max-width: 72rem`, отступы `1.5rem`.
- Брейкпоинты: `1023px`, `767px`, `575px`. Мобильное меню появляется
  до `767px` (`.leto-menu-toggle`, класс `.is-open` на `.leto-primary-nav`).
- Скругления: `0.5rem` (кнопки), `1rem` (карточки), `1.5rem` (отзывы).

## Структура файлов

### Корневые шаблоны

- `header.php` — шапка: логотип (SVG `assets/images/logo.svg` + текст «Лето»),
  теглайн «Коттеджный посёлок», навигация `primary`, телефон, CTA
  «На экскурсию» (кнопка `data-leto-modal-open="leto-tour-modal"`).
  Ниже шапки — разметка модального окна `#leto-tour-modal` с формой CF7
  «Запись на экскурсию» (ID хранится в опции `leto_starter_tour_cf7_id`).

### Модальное окно (запись на экскурсию)

- Кнопка-триггер в хедере: `data-leto-modal-open="leto-tour-modal"`.
  Модалка: `#leto-tour-modal` (`role="dialog"`, `aria-modal`), оверлей
  `.leto-modal__overlay`, диалог `.leto-modal__dialog`, кнопка закрытия
  `.leto-modal__close` (все элементы закрытия — `data-leto-modal-close`).
- Логика в `assets/js/modal.js`: открытие, закрытие по оверлею/Esc/кнопке,
  фокус-трэп, автозакрытие после успешной отправки CF7 (`wpcf7mailsent`).
  Блокировка прокрутки body через класс `leto-modal-open`.
- Форма Contact Form 7 «Запись на экскурсию» создаётся один раз в
  `functions.php` (`leto_starter_register_tour_cf7_form`, на активацию
  плагина CF7). Поля: имя, телефон, дата, комментарий. ID формы —
  через `leto_starter_cf7_tour_form_id()`.
- Стили: блок `# Modal (tour / excursion request)` в `style.css`
  (переменные `--leto-shadow-modal`, `--leto-radius`).
- `footer.php` — три колонки меню (`footer-1/2/3`) + бренд-блок с контактами
  и копирайт. Контакты (телефон, email, адрес) заданы как переменные в начале
  файла (помечены TODO для замены на реальные).
- `front-page.php` — лендинг: последовательно подключает 9 секций
  (hero → queues → lake → glamping → ecofarm → places → developer →
  testimonials → purchase). Блок `<main id="leto-primary">` с `the_content()`
  намеренно убран — контент статической страницы по дизайну не нужен.
  Hero-секция имеет `id="leto-hero"` (цель skip-to-content на главной).
- Кастомные шаблоны страниц (`page-about.php`, `page-ekofarm.php`,
  `page-usloviya.php`, `page-developer-page.php`, `page-kontakty.php`) — тоже
  НЕ выводят `the_content()` (убран `<main id="leto-primary">`). Контент
  страницы по дизайну не нужен; вся разметка — в `template-parts`. Первая
  секция каждого шаблона имеет `id="leto-hero"` для skip-to-content.
- `page.php` — базовый шаблон (для страниц без кастомного шаблона, напр.
  «Очереди»). Выводит `the_content()` в `<main id="leto-primary">`.
  `.leto-main` больше не имеет `min-height: 60vh` (убран пустой отступ
  на коротких страницах).
- `page.php` — страница (одна колонка через `.leto-layout`).
- `page-about.php` — шаблон «О посёлке»: hero-секция с заголовком
  и подзаголовком на тёмно-зелёном фоне, затем секция локации
  (`template-parts/about-location.php`), секция инфраструктуры
  (`template-parts/about-infra.php`), секция туризма
  (`template-parts/about-tourism.php`), затем контент страницы.
- `page-developer-page.php` — шаблон «Застройщикам»: hero-секция
  «Застройщик ЦМД» на тёмно-зелёном фоне, секция «О компании»
  (`template-parts/developer-page-about.php`), секция преимуществ
  (`template-parts/developer-page-features.php`), секция гарантий
  (`template-parts/developer-page-guarantees.php`), секция отзывов-CTA
  (`template-parts/developer-page-reviews.php`), затем контент страницы.
- `page-ekofarm.php` — шаблон «Экоферма»: hero-секция «Экоферма
  «Ольгино Лето»» с двухколоночным макетом (текст + изображение продуктов),
  секция «История основания» (`template-parts/ekofarm-story.php`),
  секция «Продукция и цены» (`template-parts/ekofarm-products.php`),
  секция «Подписка на Фермерский ящик» (`template-parts/ekofarm-subscribe.php`),
  секция «Мастер-классы» (`template-parts/ekofarm-masterclass.php`),
  затем контент страницы. Форма CF7 создаётся программно (опция
  `leto_ef_subscribe_form_id`).
- `page-usloviya.php` — шаблон «Условия»: hero-секция «Условия покупки»
  на тёмно-зелёном фоне, секция способов оплаты
  (`template-parts/usloviya-payment.php`), секция цен и инвестиционного
  потенциала (`template-parts/usloviya-prices.php`), CTA-секция
  (`template-parts/usloviya-cta.php`) впритык к футеру (одинаковый фон,
  `margin-top: 0` через body class `leto-page-usloviya`), затем контент
  страницы.
- `page-kontakty.php` — шаблон «Контакты»: двухколоночный макет
  (`template-parts/kontakty-contacts.php`) — левая колонка: заголовок,
  контактные данные (телефон, email, адрес), плейсхолдер карты,
  «Как добраться»; правая колонка: заголовок «Запишитесь на экскурсию»,
  форма CF7 (имя, телефон, дата, кнопка). Форма создаётся программно
  (опция `leto_kontakty_form_id`), затем контент страницы.
- `single.php` — запись (две колонки с сайдбаром `.leto-layout--with-sidebar`).
- `index.php`, `archive.php`, `search.php`, `404.php`, `comments.php`,
  `sidebar.php`, `searchform.php` — стандартные шаблоны.

### Template parts (`template-parts/`)

Секции лендинга: `hero`, `queues`, `lake`, `glamping`, `ecofarm`, `places`,
`developer`, `testimonials`, `purchase`.

Секции страницы «О посёлке»: `about-location`, `about-infra`, `about-tourism`.

Секции страницы «Застройщикам»: `developer-page-hero`, `developer-page-about`,
`developer-page-features`, `developer-page-guarantees`, `developer-page-reviews`.

Секции страницы «Экоферма»: `ekofarm-hero`, `ekofarm-story`,
`ekofarm-products`, `ekofarm-subscribe`, `ekofarm-masterclass`.

Секции страницы «Условия»: `usloviya-hero`, `usloviya-payment`,
`usloviya-prices`, `usloviya-cta`.

Секция страницы «Контакты»: `kontakty-contacts`.

Контентные части: `content.php`, `content-page.php`, `content-single.php`,
`content-search.php`, `content-none.php`.

### Функции и ассеты

- `functions.php` — регистрация поддержки темы, меню (`primary`, `footer-1/2/3`),
  сайдбара `sidebar-1`, размера `leto-starter-entry` (1200×600), подключение
  стилей и `assets/js/navigation.js`. Фолбэк футер-меню:
  `leto_starter_footer_menu_fallback`.
- `assets/js/navigation.js` — мобильное меню.
- `assets/images/` — `logo.svg`, фоны секций (`lake-bg.jpg`, `glamping-bg.jpg`,
  `ecofarm-bg.jpg`, `developer-bg.jpg`), фото мест `place-1..5.jpg`.

## Классы и нейминг

- Все CSS-классы с префиксом `.leto-` (напр. `.leto-hero`, `.leto-queue-card`).
- Все PHP-функции темы с префиксом `leto_starter_` (напр. `leto_starter_setup`).
- Все строки через `esc_html__( '...', 'leto-starter' )` (text domain
  `leto-starter`).
- Константы темы: `LETO_STARTER_VERSION`, `LETO_STARTER_DIR`, `LETO_STARTER_URI`.

## Поведение секций (кратко)

- **Hero** — тёмно-зелёный фон, высота `80vh`, видео-плейсхолдер + заголовок
  справа, две кнопки (primary белая / outline).
- **Очереди (queues)** — кремовый фон, сетка из 4 карточек с ценой/бейджем.
- **Озеро (lake)** — фоновое фото `lake-bg.jpg` + зелёный оверлей, статистика.
- **Глэмпинг (glamping)** — белый фон, фото слева + текст/список справа.
- **Экоферма (ecofarm)** — кремовый фон, форма (чёрные инпуты) + радио-группа.
- **Места (places)** — белый фон, сетка карточек с фото и номером.
- **Застройщик (developer)** — кремовый фон, блок + 3 фичи-карточки (синий
  заголовок).
- **Отзывы (testimonials)** — тёмно-зелёный фон, карусель (JS-навигация
  стрелками/точками).
- **Покупка (purchase)** — белый фон, 3 карточки.
- **О посёлке (about-hero)** — тёмно-зелёный фон, центрированный заголовок
  и подзаголовок. Используется в шаблоне `page-about.php`.
- **Локация и природа (about-location)** — белый фон, двухколоночный макет
  (текст слева + изображение справа), eyebrow «Локация и природа»,
  заголовок, описание, сетка 2×2 со статистикой (время/расстояние).
  Адаптив: на мобильных изображение перемещается наверх.
- **Инфраструктура (about-infra)** — кремовый фон, центрированный заголовок,
  сетка из 3 карточек (иконка + синий заголовок + описание). Адаптив:
  на планшетах 2 колонки, на мобильных 1 колонка.
- **Туристический кластер (about-tourism)** — белый фон, центрированный
  заголовок, сетка 2×2 с карточками (название + бейдж статуса + описание).
  Бейджи: Активно (зелёный), Строится (оранжевый), Реставрация (синий),
  Проект (фиолетовый). Адаптив: на мобильных 1 колонка.
- **Застройщикам — Hero (dp-hero)** — тёмно-зелёный фон, центрированный
  заголовок «Застройщик ЦМД» и подзаголовок. Паттерн аналогичен about-hero.
- **Застройщикам — О компании (dp-about)** — белый фон, двухколоночный макет
  (текст слева + изображение справа), заголовок, два абзаца описания, сетка
  3×1 со статистикой (2008 / 500+ / 5 лет). Адаптив: на мобильных 1 колонка.
- **Застройщикам — Преимущества (dp-features)** — кремовый фон, сетка 3×2
  из карточек (иконка + синий заголовок + описание). 6 карточек:
  15 лет на рынке, Гарантия 5 лет, Более 500 домов, Команда профессионалов,
  Сроки без задержек, Под ключ. Адаптив: на мобильных 1 колонка.
- **Застройщикам — Гарантии (dp-guarantees)** — кремовый фон, центрированный
  заголовок «Гарантии и документы» + подзаголовок, список из 5 пунктов
  с иконкой-галочкой в белых прямоугольниках. Максимальная ширина 48rem.
- **Застройщикам — Отзывы CTA (dp-reviews)** — тёмно-зелёный фон, центрированный
  заголовок «Нам доверяют более 100 семей», описание, кнопка «Читать отзывы».
  Паттерн аналогичен testimonials, но упрощённый (без карусели).
- **Экоферма — Hero (ef-hero)** — тёмно-зелёный фон, двухколоночный макет
  (текст слева + изображение продуктов справа), eyebrow «Экоферма»,
  заголовок «Экоферма «Ольгино Лето»», описание, цитата. Адаптив:
  на мобильных 1 колонка.
- **Экоферма — История (ef-story)** — белый фон, центрированный заголовок
  «История основания», описание, сетка 3×1 с карточками направлений
  (Растениеводство, Птицеводство, Семейная пекарня). Адаптив:
  на мобильных 1 колонка.
- **Экоферма — Продукция (ef-products)** — кремовый фон, центрированный
  заголовок «Продукция и цены», список из 6 товаров с иконками и ценами
  в белом контейнере с разделителями.
- **Экоферма — Подписка (ef-subscribe)** — белый фон, центрированный блок
  с иконкой-корзиной, заголовок «Подписка на «Фермерский ящик»», форма CF7
  (имя, телефон, радио-группа периодичности, кнопка). Форма создаётся
  программно, ID хранится в опции `leto_ef_subscribe_form_id`.
- **Экоферма — Мастер-классы (ef-masterclass)** — кремовый фон, центрированный
  заголовок «Учитесь у Ольги», описание, карточка «Ближайшие мастер-классы»
  с иконкой-календарём и текстом-заглушкой.
- **Условия — Hero (usl-hero)** — тёмно-зелёный фон, центрированный заголовок
  «Условия покупки» и подзаголовок. Паттерн аналогичен about-hero.
- **Условия — Способы оплаты (usl-payment)** — белый фон, сетка 3×1
  с карточками (зелёная иконка, синий заголовок, описание, список
  с галочками). Карточки: Рассрочка от застройщика, Ипотека,
  Юридическое сопровождение. Адаптив: на мобильных 1 колонка.
- **Условия — Цены (usl-prices)** — кремовый фон, центрированный заголовок
  «Цены и акции», таблица с тёмно-зелёной шапкой (4 очереди, цены,
  цели, потенциал), цитата-блок внизу.
- **Условия — CTA (usl-cta)** — тёмно-зелёный фон (впритык к футеру,
  body class `leto-page-usloviya` убирает `margin-top` футера), заголовок
  «Готовы сделать шаг к своему дому?», белая кнопка «Получить консультацию».
- **Контакты (kontakty-contacts)** — белый фон, двухколоночный макет.
  Левая колонка: заголовок «Как с нами связаться», контактные данные
  (телефон, email, адрес) с иконками, плейсхолдер карты, блок
  «Как добраться» (машина, автобус, трансфер). Правая колонка:
  заголовок «Запишитесь на экскурсию», форма CF7 на кремовом фоне
  (имя, телефон, дата, кнопка «Записаться на экскурсию» с иконкой
  самолётика, consent-текст). Адаптив: на мобильных 1 колонка.

## Что учитывать при доработках

- Новые секции лендинга добавлять как `template-parts/<name>.php` и
  подключать в `front-page.php` через `get_template_part()`.
- Новые цвета/шрифты заводить через CSS-переменные `:root` в `style.css`.
- Все изменения структуры/дизайна сразу отражать в этом файле (см. AGENTS.md,
  раздел «Обновление документации»).
